Tool

Draw: click to drop a node onto the selected branch.
S cycle waypoint → station → minor stop · Z undo node · B new branch
Move: drag a node. Del removes selected node, S cycles its type (waypoint / station / minor).
Pan/zoom: scroll or pinch to zoom, drag (or hold Space) to pan.

Lines

Color comes from the line name (same palette as the app). Pick a type, then + Line. Changing the dropdown recolors the selected line.

Branches

A line can hold several branches (e.g. Green B/C/D/E, Red Ashmont/Braintree). Fork seeds a new branch at the selected node so the trunk is shared.

Label stations

Click a station to name it, or load the CSV to bulk-fill.
